Class Activity: Random Walks Continued (15 min)

Recall the definition of a random walk:

\(\{x_t\}\) is a random walk if it can be expressed as \[ x_{t} = x_{t-1} + w_{t} \] where \(\{w_t\}\) is a white noise series.

There are other ways to represent this expression.

Check Your Understanding

  • Notice that \(x_{t-1} = x_{t-2} + w_{t-1}\) and \(x_{t-2} = x_{t-3} + w_{t-2}\). Use this to explain why it is possible to write \(x_t\) as \[ x_{t} = \sum\limits_{i=-\infty}^{t} w_{i} = \cdots + w_{t-3} + w_{t-2} + w_{t-1} + w_{t} \] where \(\{w_t\}\) is a DWN time series.

    If the time series is finite, we can write \(x_t\) as: \[ x_{t} = w_1 + w_2 + w_3 + \cdots + w_{t-3} + w_{t-2} + w_{t-1} + w_{t} \]

Definition of the Backward Shift Operator

This process of back substitution is so common, we define notation to handle it.

Definition of the Backward Shift Operator

We define the backward shift operator or the lag operator, \(\mathbf{B}\), as: \[ \mathbf{B} x_t = x_{t-1} \] where \(\{x_t\}\) is any time series.

We can apply this operator repeatedly. We will use exponential notation to indicate this.

\[ \mathbf{B}^2 x_t = \mathbf{B} \mathbf{B} x_t = \mathbf{B} ( \mathbf{B} x_t ) x_t = \mathbf{B} x_{t-1} = \mathbf{B} x_{t-2} \]

In general, \[ \mathbf{B}^n x_t = \underbrace{\mathbf{B} \cdot \mathbf{B} \cdot \cdots \cdot \mathbf{B}}_{n ~ \text{terms}} x_t = \mathbf{B}^{n-1} ( \mathbf{B} x_t ) = \mathbf{B}^{n-1} ( x_{t-1} ) = \mathbf{B}^{n-2} ( x_{t-2} ) = \cdots = \mathbf{B} x_{t-(n-1)} = x_{t-n} \]

Note that we can use the backwards shift operator to write a random walk in terms of the realizations of a white noise process. Consider the following process: \[\begin{align*} x_t &= \frac{1}{5} w_{t-2} + \frac{1}{3} w_{t-1} + w_{t} \\ &= \frac{1}{5} \mathbf{B}^2 w_t + \frac{1}{3} \mathbf{B} w_t + w_{t} \\ &= \left( \frac{1}{5} \mathbf{B}^2 + \frac{1}{3} \mathbf{B} + 1 \right) w_{t} \\ \end{align*}\]

We will practice applying this operator.

Second-Order Properties of a Random Walk

The second-order properties of a random walk are summarized below.

Second-Order Properties of a Random Walk

If \(\{x_t\}_{t=1}^n\) is a random walk, then the population has the following properties.

\[ \mu_x = 0 \] and \[ cov(x_t, x_{t+k}) = t \sigma^2 \]

Click here for a proof of the equation for \(cov(x_t,x_{t+k})\)

Why is \(cov(x_t, x_{t+k}) = t \sigma^2\)?

First, note that that since the terms in the white noise series are independent,

\[ cov ( w_i, w_j ) = \begin{cases} \sigma^2, & \text{if } ~ i=j \\ 0, & \text{otherwise} \end{cases} \]

Also, when random variables are independent, the covariance of a sum is the sum of the covariance.

Hence, \[\begin{align*} cov(x_t, x_{t+k}) &= cov ( \sum_{i=1}^t w_i, \sum_{j=1}^{t+K} w_j ) \\ &= \sum_{i=j} cov ( w_i, w_j ) \\ &= \sum_{i=1}^t \sigma^2 \\ &= t \sigma^2 \end{align*}\]

If \(k>0\) and \(t>0\), the correlation function is

\[ \rho_k = \frac{ cov(x_t, x_{t+k}) }{ \sqrt{var(x_t)} \sqrt{var(x_{t+k})} } = \frac{t \sigma^2}{\sqrt{t \sigma^2} \sqrt{(t+k) \sigma^2}} = \frac{1}{\sqrt{1+\frac{k}{t}}} \]

Note that the covariance of a random walk process depends on \(t\). Hence, random walks are non-stationary. The variance is unbounded as \(t\) increases. That implies a random walk will not provide good predictions in the long term.

Note that if \(0 < k \ll t\), then \(\rho_k \approx 1\). Because of this, a correlogram for a random walk will typically demonstrate positive autocorrelations that start near 1 and slowly decrease as \(k\) increases.

Differencing a Time Series

Computing the difference between successive terms of a random walk leads to a discrete white noise series. In many cases, differencing sequential terms of a non-stationary process can lead to a stationary process of differences. Defintion of the Difference Operator

Differencing nonstationary time series often leads to a stationary series, so we will define a formal operator to express this process.

Definition of the Difference Operator

The difference operator, \(\nabla\), is defined as:

\[\nabla x_t = x_t - x_{t-1} = (1-\mathbf{B}) x_t\]

Higher-order differencing can be denoted

\[\nabla^n x_t = (1-\mathbf{B})^n x_t\]

To see what this expression gives us, note that \(\nabla\) gives a new time series that is comprised of the differences between successive terms of the original time series. The operator \(\nabla^2\) generates a time series that is comprised of the differences between successive terms of the differenced time series. It is the difference of the differences, or the second difference.

Class Activity: Random Walk with Drift (15 min)

# Set date range
date_start <- "2023-05-01"
date_end <- "2024-01-01"

We will now consider the daily closing price of Abercrombie & Fitch stock (Symbol = ANF). Here is a time series plot of the closing stock prices from d May 2023 to 01 Jan 2024.

# Set symbol 
symbol <- "ANF"

# Fetch stock prices
df_stock <- tq_get(symbol, from = date_start, to = date_end, get = "stock.prices")

# Transform data into tibble
df_tsibble <- df_stock |>
  mutate(
    dates = date, 
    value = close
  ) |>
  dplyr::select(dates, value) |>
  as_tibble() |> 
  arrange(dates) |>
  as_tsibble(index = dates, key = NULL)

# Generate time series plot using plot_ly
plot_ly(df_tsibble, x = ~dates, y = ~value, type = 'scatter', mode = 'lines') |>
  layout(
    xaxis = list(title = "Date"),
    yaxis = list(title = "Value"),
    title = paste0("Time Plot of ", symbol, " Daily Closing Price")
  )

We now generate a time plot and a correlogram of the differences.

Show the code 
df_tsibble$diff = df_tsibble$value - lag(df_tsibble$value) 
df_tsibble |>
  na.omit() |>

        paste0(
          format(min(df_tsibble$dates), "%d %b %Y"),
            " - ",

      plot.title = element_text(hjust = 0.5),
      plot.subtitle = element_text(hjust = 0.5)
    )

Show the code 
acf(df_tsibble$diff |> na.omit(), main = "ACF of First Difference of ANF Stock Prices")

There is no significant autocorrelation in the differences. They appear to be modeled well by white noise.

We now compute the mean and standard deviation of the differences.

mean_diff <- df_tsibble$diff |> mean(na.rm = TRUE)
sd_diff <- df_tsibble$diff |> sd(na.rm = TRUE)
n_diff <- df_tsibble$diff |> na.omit() |> length()

The mean of the differences is 0.386. The standard deviation of the differences is 1.51. There are 168 differences.

We can use the t-distribution to create a 95% confidence interval for the drift parameter. The critical \(t\) value is given by qt(0.975, df = 168 - 1), yielding a value of \(t^*_{0.975} = 1.974\).

So, our 95% confidence interval is computed as: \[ \left( \bar x - t^*_{0.975} \cdot \frac{s}{\sqrt{n}} , ~ \bar x + t^*_{0.975} \cdot \frac{s}{\sqrt{n}} \right) \] \[ \left( 0.386 - 1.974 \cdot \frac{1.51}{\sqrt{168}} , ~ 0.386 + 1.974 \cdot \frac{1.51}{\sqrt{168}} \right) \] \[ (0.156 , ~ 0.616) \]

This confidence interval does not contain 0, so we conclude that there is evidence of a positive drift in the price of Abercrombie & Fitch stock over this period.

Difference Operator

\(x_{1} = 5\),\(~\) \(x_{2} = 10\),\(~\) \(x_{3} = 13\),\(~\) \(x_{4} = 8\),\(~\) \(x_{5} = 4\),\(~\) \(x_{6} = 3\),\(~\) \(x_{7} = 9\),\(~\) \(x_{8} = 2\)

Check Your Understanding Solutions:

  • The values of the first differences are: \[ \nabla x_t = \{ 5, ~~ 3, ~~ -5, ~~ -4, ~~ -1, ~~ 6, ~~ -7\}\]

  • The values of the second differences are: \[ \nabla^2 x_t = \{ -2, ~~ -8, ~~ 1, ~~ 3, ~~ 7, ~~ -13\}\]

  • By substitution, we get:

\[\begin{align*} (1-\mathbf{B} )^2 x_8 &= (1-\mathbf{B} ) \left[ (1-\mathbf{B} ) x_8 \right] \\ &= (1-\mathbf{B} ) ( x_8 - x_7 ) \\ &= ( x_8 - x_7 ) - \mathbf{B} ( x_8 - x_7 ) \\ &= ( x_8 - x_7 ) - ( x_7 - x_6 ) \\ &= ( \nabla x_8 ) - ( \nabla x_7 ) \\ &= \nabla ( \nabla x_8 ) \\ &= \nabla^2 x_8 \end{align*}\]

\[ \nabla^2 x_8 = ( x_8 - x_7 ) - ( x_7 - x_6 ) = (2 - 9) - (9 - 3) \\ = -13 \] This is the value of the last term in the sequence of second differences.
